# Linkedin-Auto-Meesage-and-Connect-Sender

## How to work

## 1. Install python
```
https://www.python.org/downloads/
```

### 2. install requirement.txt files

```
$ pip install -r /path/to/requirements.txt
```

## 3. Give name a flask app
``` 
$ export FLASK_APP=hello
```

## 4. Run flask
``` 
$ flask run
```
